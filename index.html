üöÄ How to Upload a Static Web App to Azure

Azure Static Web Apps is a service that automatically builds and deploys full-stack web apps to Azure from a GitHub repository.

Steps to Deploy:

  1. Create a GitHub repository with your static site (e.g., this index.html file).
  2. Push your code to GitHub.
  3. Go to the Azure Portal and search for Static Web Apps.
  4. Click + Create, fill in your resource details.
  5. Connect your GitHub account and select the repository and branch.
  6. Azure will set up a GitHub Action to build and deploy your site automatically.
  7. After deployment, you'll get a live URL for your static website.
